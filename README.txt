A Pen created at CodePen.io. You can find this one at https://codepen.io/mtHuberty/pen/bogMJd.

 Makes a call to the wikipedia api based on the search term that is entered. If no search term is entered, the app opens a random article.

Results and search bar animate when hovered over or clicked onusing css effects. If you enter a search term that returns no results, an alert box will you inform you that your search returned no results.
